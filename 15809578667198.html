isset

判断变量是否已配置。



语法: int isset(mixed var);



返回值: 整数



函数种类: PHP 系统功能



内容说明 



本函数用来测试变量是否已经配置。若变量已存在则返回 true 值。其它情形返回 false 值。

使用范例 



<?php

\(a = "test";

echo isset(\)a); // true

unset(\(a);

echo isset(\)a); // false

?>
